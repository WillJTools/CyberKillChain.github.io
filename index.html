Understanding the Cyber Kill Chain

1. Reconnaissance

Example: An attacker uses public sources like social media, company websites, and LinkedIn profiles to gather information about employees' roles and responsibilities. This information helps them craft convincing spear-phishing emails tailored to specific individuals.

Tools: theHarvester, Shodan, Recon-ng

2. Weaponization

Example: An attacker attaches a malicious macro to an innocuous-looking Word document. When the victim opens the document and enables macros, the malware is executed, allowing the attacker to gain access to the victim's system.

Tools: Metasploit, Cobalt Strike, Social Engineer Toolkit (SET)

3. Delivery

Example: An attacker sends a phishing email to an employee, pretending to be from a legitimate source like the HR department. The email contains a link that leads to a fake login page where the victim unwittingly enters their credentials.

Tools: GoPhish, SET, Maltego

4. Exploitation

Example: An attacker identifies a known vulnerability in a web server software that the target company is using. They exploit this vulnerability using a crafted HTTP request to gain unauthorized access to the server.

Tools: Metasploit, Burp Suite, Nessus

5. Installation

Example: Once inside the system, the attacker uploads a backdoor—a malicious software that allows remote access—into a seemingly harmless application that employees commonly use.

Tools: Netcat, Meterpreter, Empire

6. Command and Control (C2)

Example: The attacker uses a compromised system to communicate with a remote server, disguising the communication as legitimate traffic to avoid detection by security measures.

Tools: Cobalt Strike, Metasploit, Covenant

7. Execution

Example: The attacker uses the compromised system to harvest sensitive data, such as credit card information, usernames, and passwords, which are then exfiltrated to a remote server controlled by the attacker.

Tools: Mimikatz, BloodHound, CrackMapExec

8. Actions on Objectives

Example: With the stolen data in hand, the attacker attempts to sell the information on underground forums, potentially leading to financial loss and reputation damage for the targeted organization.

Tools: Tor, Darknet forums, Cryptocurrencies for transactions
